# 仕様
  - カレンダーは月単位で切り替えできるようにします。
  - カレンダーの日付をクリックすると日記が編集できるようにしてください。
  - カレンダーには日記の内容を文字で表示してください。
  - 日記データはjsonで保存します。
  
# 依頼する作業

  - Unitテスト
  
# その他
  - 開発環境は Windows です 
  - フロントエンドはJavascript、バックエンドはPython flaskで作成してください。
  - ユーザはPCでWEBブラウザを用いて利用します。見やすいレイアウトでお願いします。


  - ５週ある月は日、月、火、水、木、金、土 の順番で５段で表示します。
